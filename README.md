👑 **Admin Account**: Has full control over the system. The navigation menu is fully visible, including the Account Management module for adding, editing, and deleting personnel records.

## 📥 Installation

### 1. Clone Repository

```bash
git clone https://github.com/NV-DuyManh/CINEMA_GROUP6.git
```

### 2. Open Project

Open the file:

```
Cinema.sln
```

using **Visual Studio (2017/2022)**.

---

## 🗄 Database Setup
1. Open **Server Explorer** in Visual Studio
2. Right-click **Data Connections → Add Connection**
3. Select:

```
Microsoft SQL Server Database File
```

4. Browse to:

```
Cinema/bin/Debug/DBRapPhim.mdf
```

5. Click **Open → OK**

---

## ▶ Run Application

Press **F5** in Visual Studio to start the program.
